Wolfram Cellular Automaton in pygame
A graphical interface for simulating Wolfram's cellular automata with customizable rules and parameters.

Features:
- Start/stop simulation with a button.
- Input fields for rule number (0-255) and cell size.
- Toggle between rainbow and monochrome colors.
- Real-time visualization of automaton evolution.

How to Use:
1. Run the Program:
   >>python main.py
2. Customize Parameters:
   - Enter a rule number (0-255).
   - Set cell size.
   - Use the checkbox to enable/disable rainbow colors.
3. Control Simulation:
   - Click "Start" to begin or "Stop" to pause.
   - Adjust parameters while paused.

Requirements
- Python 3.x
- Pygame library

Installation:
1. Download the source files.
2. Install dependencies:
   >>pip install pygame

Notes:
- Grid size adapts to window dimensions and cell size.
- Invalid rule numbers default to 82.
